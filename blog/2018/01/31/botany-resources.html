Botany resources

Plants are wonderful. I’ve created some botany resources which others might find useful. I plan to add more to this page in the future. I’m interested in working on vector graphics especially. You may email me with any corrections.

Plant Latin Dictionary

I adapted Jason Hollinger’s Plant Latin Dictionary into a 5.5' × 8.5" booklet. The booklet is 20 pages, which fits onto five sheets. The file is a 200 KB PDF.

Common Plants of Wisconsin flashcards

I adapted UW–Stevens Point’s Common Plants of Wisconsin image collection into a set of 5.5' × 8.5" color flashcards. The file is a 15 MB PDF.
